# JavaDSA

## Data Structures
Java Collections Framework (JCF)

```
Collection<E>
    |-- List<E>
    |    |-- ArrayList<E>
    |    |-- LinkedList<E>
    |    |-- Vector<E>
    |         |-- Stack<E>
    |
    |-- Set<E>
    |    |-- HashSet<E>
    |    |-- LinkedHashSet<E>
    |    |-- SortedSet<E>
    |         |-- NavigableSet<E>
    |              |-- TreeSet<E>
    |
    |-- Queue<E>
         |-- LinkedList<E>
         |-- PriorityQueue<E>
         |-- Deque<E>
              |-- ArrayDeque<E>

Map<K, V>
    |-- HashMap<K, V>
    |-- LinkedHashMap<K, V>
    |-- SortedMap<K, V>
         |-- NavigableMap<K, V>
              |-- TreeMap<K, V>
    |-- Hashtable<K, V>
```
